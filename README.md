# MyMovies

Aplicativo React Native para exibir detalhes de filmes usando TMDB e OMDb.

## Como rodar

1. Instale as dependências:
   ```
   npm install
   ```
2. Configure as chaves de API no arquivo .env
3. Rode o projeto:
   ```
   npx expo start
   ```

## Funcionalidades
- Busca de filmes
- Exibição de detalhes (elenco, diretores, avaliações, prêmios, país de produção)
- Imagens de bandeiras dos países

## Tecnologias
- React Native
- Expo
- TMDB API
- OMDb API
